# demo-express-api
This is a course demo for Express API with Postgres and Jest

## Run locally
- Use Node v16
- Install packages
```
npm install
```
- Start project
```
npm start
```
- Try `http://localhost:6065` to test the root route

## Run unit-tests
```
npm run test
```

## Run API test
- Start the project for ready to the API tests calling
```
npm start
```
- Open another Tap of Terminal and run:
```
npm run apitest
```
